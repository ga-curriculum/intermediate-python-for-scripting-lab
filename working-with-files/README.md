  <span class="subhead">Working With Files</span>
</h1>

**Learning objective:** By the end of this lesson, students will be able to manipulate files using Python.

## Setup

Create a new file named <code class="filepath">files.py</code> in your <code class="filepath">intermediate-python-for-scripting-lab</code> directory. This file will contain all the code for this exercise.

Create a new text file named <code class="filepath">sample.txt</code> in the same directory. Add a few words to this file, but don't worry too much about the content. Consider using an AI assistant to generate some text if you want to add content to it quickly.

## Exercises

1. Create a function with a logical name of your choice. It should:
   - Read a text file.
   - Counts the number of words in the text file.
   - Handle potential `FileNotFoundError` exceptions gracefully.

2. Write a function with a logical name of your choice. It should:
   - Take a list of strings.
   - Write each string to a new line in a file named <code class="filepath">output.txt</code>.
