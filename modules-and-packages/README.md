**Learning objective:** By the end of this lesson, students will be able to use modules to solve real world problems.

# Exercises

1. Create a function that generates a random password of a specified length using the `random` module.  The password should include a mix of uppercase letters, lowercase letters, and digits.

2. Use the datetime module to write a function that calculates the number of days between two given dates (provided as strings in "YYYY-MM-DD" format).

3. From the `math` module, import only the `sqrt` and `pi` functions.  Write a function that calculates the area of a circle using these imported functions.
