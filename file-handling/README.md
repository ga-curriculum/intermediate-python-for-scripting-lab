**Learning objective:** By the end of this lesson, students will be able to manipulate files using Python.

# Exercises

1. Write a function that reads a text file and counts the number of words in it.  Handle potential `FileNotFoundError` exceptions gracefully.

2. Write a function that takes a list of strings and writes each string to a new line in a file named "output.txt".

3. Write a function that reads a CSV file (you can create a sample CSV) and prints the contents of each row.  Use the `csv` module.
