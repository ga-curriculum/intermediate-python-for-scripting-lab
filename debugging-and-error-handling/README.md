**Learning objective:** By the end of this lesson, students will be able to use try-except blocks and raise custom errors.

# Exercises

1. You are given some code that attempts to divide two numbers.  However, it's encountering a `ZeroDivisionError`.  Use a try-except block to handle this error and print an informative message.

2. Create a function that validates an email address. If the email address does not contain an "@" symbol, raise a custom exception called `InvalidEmailError`.
