## Setup

Open your Terminal application and navigate to your <code class="filepath">~/code/ga/labs</code> directory:

```bash
cd ~/code/ga/labs
```

### Create and clone a GitHub repo

Make a new repository on [GitHub](https://github.com/) named `intermediate-python-for-scripting-lab`.

Clone a copy of your remote repo locally by using the `git clone` command:

```bash
git clone https://github.com/<your-username>/intermediate-python-for-scripting-lab.git
```

Replace `<your-username>` (including the `<` and `>`) with your GitHub username before you run this command.

### Launch in VS Code

Next, `cd` into the cloned directory, <code class="filepath">intermediate-python-for-scripting-lab</code>:

```bash
cd intermediate-python-for-scripting-lab
```

Open the contents of the directory in VS Code:

```bash
code .
```
